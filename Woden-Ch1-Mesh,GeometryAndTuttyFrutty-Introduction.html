The goal of this chapter is to explain the basics of OpenGL graphics primitives, geometry and Basic structure.

II. Graphics Primitives

By default OpenGL renders object in combinaison of point, lines (undertanding line segments) or triangles. Points are single vertex (a three-dimensional points in space). OpenGL provide some function to set the pixel value of a point, it is important to notice that each pixel of the point will be pass throught the fragment shader (so a point can have multiple color dependinhg how many pixel is allocated to it).
